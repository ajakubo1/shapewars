========
Overview
========

This is overwiew of shapewars application

----------
Map object
----------

* 0-7 - values reserved for players
* 8 - square which is unavailable to reach
* 9 - suqare availible to be conquered

-------------
Player object
-------------

* ``name`` - player name
* ``color`` - player color
* ``type`` - player type:

    - 0 - current player
    - 1 - enemy human
    - 2 - enemy AI
